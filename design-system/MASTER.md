# X Circle Operator Design System

> Generated: 2026-05-29
> Last updated: 2026-05-29

## Direction

**Style**: Quiet operational console  
**Tone**: Privacy-aware, restrained, and built for repeated review work rather than marketing.

## Colors

| Purpose | Color |
| --- | --- |
| Background | `#f3f1ea` |
| Surface | `#fffcf4` |
| Ink | `#171612` |
| Muted | `#6e6a5f` |
| Line | `#d8d1c1` |
| Accent | `#0f6f5f` |
| Warning | `#9b6a16` |
| Danger | `#a33030` |

## Typography

- **Display**: Space Grotesk
- **Body**: Geist

## Interface Rules

- Dense dashboard first, no landing-page hero.
- Cards only for repeatable lead/task rows.
- Buttons use lucide icons where possible.
- All status-changing actions remain explicit buttons.
- No automated-send affordance or language.
- Radius: 8px for controls and rows.
- Motion: 180ms micro-interactions, disabled under `prefers-reduced-motion`.
